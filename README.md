# 💰 Finance Dashboard UI

## 📌 Overview

A modern and interactive **Finance Dashboard UI** built using React.
This project allows users to track financial data, analyze spending patterns, and manage transactions with a clean and responsive interface.

---

## 🚀 Live Demo

🔗 https://muskan-tuteja.github.io/Finance-Dashboard-UI/

---

## ✨ Features

### 📊 Dashboard Overview

* Total Balance, Income, Expenses summary cards
* Clean and minimal UI design

### 📈 Data Visualization

* Line Chart for balance trends
* Pie Chart for spending breakdown

### 💳 Transactions Management

* View all transactions
* Search by category
* Filter by Income / Expense
* Add new transactions
* ✏️ Edit existing transactions (Admin only)
* 🔄 Real-time UI updates

### 🔐 Role-Based UI

* **Viewer**: Can only view data
* **Admin**: Can add new transactions

### ➕ Add Transactions

* Dynamic form to add new data
* Updates UI instantly

### 💡 Insights

* Highlights highest spending category
* Provides useful financial observations

## ▶️ Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/muskan-tuteja/Finance-Dashboard-UI.git
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the project

```bash
npm run dev
```
