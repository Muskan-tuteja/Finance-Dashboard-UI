# 💰 Finance Dashboard UI

## 📌 Overview

This is a frontend finance dashboard built using React. It allows users to track financial data, view transactions, and analyze spending patterns.

## 🚀 Features

* Dashboard summary (Balance, Income, Expenses)
* Charts (Line & Pie)
* Transactions table with search & filter
* Role-based UI (Admin / Viewer)
* Add transactions (Admin only)
* Insights section

## 🛠️ Tech Stack

* React.js
* Tailwind CSS
* Recharts

## ▶️ Run Project

```bash
npm install
npm run dev
```
